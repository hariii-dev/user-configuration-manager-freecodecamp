# User Configuration Manager 🛠️

A simple, interactive Python program to manage user configuration settings (like themes, language, notification preferences, etc.).

## Features
- **Add Settings:** Safely add new settings with lowercase conversion.
- **Update Settings:** Easily modify existing settings.
- **Delete Settings:** Remove unwanted settings smoothly.
- **View Settings:** Beautifully formatted and capitalized user settings display.

## How to Run
Run the Python script using:
```bash
python main.py
